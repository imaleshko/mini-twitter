# Mini Twitter

Невелика соціальна мережа, розроблена за допомогою Ruby on Rails.

## Основний функціонал

* **Аутентифікація**: Реєстрація та вхід.
* **Стрічка**: Створення коротких постів та перегляд загальної стрічки.
* **Взаємодія між користувачами**: Можливість переглядати список користувачів та система підписок.
* **Приватні повідомлення**: Чати між користувачами.

## Технології

Ruby on Rails 8
